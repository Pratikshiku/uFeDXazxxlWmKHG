## College Student Internship Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/68e5db24d3c54a9cb114b5ecdf0c2706/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): student users, teacher users, administrators.

**Student Features:**
1. Login
2. Registration
3. Homepage
4. Forum
5. Announcements
6. Internship Sharing
7. Personal Center
8. Background Management (Homepage, Personal Center, Forum Management, Announcement Management, Internship Management, Internship Grade Management, Internship Sharing Management, Internship Weekly Report Management)


**Teacher Features:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Center)
4. Forum Management
5. Announcement Management
6. Internship Management
7. Internship Grade Management
8. Internship Sharing Management (Internship Sharing Management, Internship Sharing Comment Management)
9. Internship Weekly Report Management


**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Post Type Management, Announcement Type Management, Unit Nature Management, Sharing Type Management, Week Management, Department Management)
4. Forum Management
5. Announcement Management
6. Teacher Management
7. Internship Management
8. Internship Grade Management
9. Internship Sharing Management (Internship Sharing Comment Management, Internship Sharing Collection Management)
10. Internship Weekly Report Management
11. Student Management
12. Carousel Image Information
